What is VisualDEP?

VisualDEP is webserver developed in the Laboratorio de Biofísica Molecular of Universidad de Concepción. VisualDEP aims to compare two highly similar proteins with respect to their electrostatic properties. This comparision is done calculating the difference in electrostatic potential at atomic level of both proteins.

How it works?

VisualDEP bases its work on a workflow which is built of four major steps:

  1. Retrieval & alignment of the sructures
  2. Calculation of electrostatic potentials
  3. Calculation of differences in electrostatic potentials
  4. Search and visualization of mayor differences in electrostatic potential

Retrieval & alignment of the structures

 At this step, the systems asks to the user for two pdb files corresponding to two highly similar protein structures. Then VisualDEP proceeds to make a structural alignment of the structures using PyMOL in background. To proceed to the next step, the RMSD resulting from the alignment must be lower or equal to 3Å, otherwise workflow aborts.




Calculation of electrostatic potentials
Electrostatic potentials are calculated from both aligned structures, this is done using PDB2PQR (with AMBER as force field) to prepair the files, and APBS to calculate the electrostatic potentials.

Calculation of differences in electrostatic potentials
Using the electrostatic potentials from both structures, an absolute difference is calculated, this is done subtracting each point from the resulting grid of each structure and the result replace the corresponding potential value for each point in the grid. Using the APBS multivalue tool with the difference in electrostatic potential done before, the potential is calculated to each atom in the structure, and then the obtained values are assigned in the charge field in the pqr file resulting from the referenced pdb of the previous step.


Search and visualization of mayor differences in
electrostatic potential
A Jmol applet is used to visualizate the difference in the reference structure, using a color gradient scheme to represent the grade of difference, with the possibility to highlight the 20% to 80% atoms with greater changes.

Using an outlier detection technique, we aim to find those values that represents highest differences in electrostatic potential, these values are presented on a table with the possibility to highlight, label and center the visualization in each of the outliers.

Input

The server require the user provide for two pdb files which represents the structures to compare, also an e-mail address is required to alert about the status of the job.

Output

A Jmol applet is used to visualize the electrostatic differences from a modified pqr file (from the first structure provided) where the charge field from each atom is replaced by the corresponding value, allowing a friendly visual interpretation. Also, the user can retrieve all resulting files from each workflow step, as well a modified pdb file where the B-factor field from each atom is replaced by the corresponding electrostatic potential differences. And a dx file corresponding to the grid of the difference in electrostatic potential calculated.
